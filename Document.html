Summary

This interactive CV allows a potentional employer read my cv as they please. By using scrollers, collapse and toggle pannels a user can expand and close information as they please. Download my CV using the Download Resume link.
Social Buttons allow vistors to find me on different social media sites such as github, twitter and linkedin. When a user clicks a link, the page will be opened in a seperate page.
See my work experience details by clicking on the panel of their choice on the workexperience timeline and see educational results by clicking on the schools image.
Check out my projects by hovering over the images and don't forget to message me using the chatbox!

About Container


Halo and Navigation

  • I implemented a scrollable feature using bootstrap's data-spy scroll, this will allow a vistor to navigate and find information easier.

  • Using bootstrap and javascript, I have made my halo image interactive by mouse on move.

  • Animated collapse scrollable arrow: Using data-toggle collapse and bootstrap feature arrow bounce and fa fa-arrow-down and moz animation.

  • Using iFrame and google maps api, I have embeded a google map into my form

Resume Container

Bootstrap Features I have implemented into resume:

  • Glyphicons: Used glyphicon images to create a nicer visualisation.

  • Timeline:Created a timeline of workexperience using bootstrap feature timeline, css and JS. Using Javascript toggle and display:none. A viewer can see more information about my workexperience by clicking on the panel they wish to view.

  • Education image click: Contains information about college and secondary education. The related image's are connected to a model which displays a table with the relevant results.

  • Created Aninated Skill Bars:Using bootstraps canvas feature I created an animated doughnut skills bar using javascript.

  • Carousel to show achievements with captions giving a quick explaination of the award/achievement.

Projects

Projects Container + carasoul : Using bootstrap hovereffect and a model, I have placed 3 images into the container that represent first, second and third year projects. When a user hovers over the image it will display the year that project belongs too. When you click on the banner a pop-up page will appear with information about the project and a carasoul of project images are available on 2nd and 3rd year projects.

Message Form: Created a "Chatbox" form that allows visitors to send an email to me using enformed and jQuery for posting methods, styled as an instant chat message-box.
